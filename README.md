# RecipeApp

Get Working with given app

To run on your local machine 🖥️ follow : 
clone git repository via link 🔗 https://github.com/ajayKumbhar/ng-recipe-book

open terminal 
cd ng-recipe-book
run npm install --save-dev angular-cli@latest
npm install
npm init

then open ng serve 

open browser
http://localhost:4200/
